Maxwell

This is Maxwell's daemon, an application that processes MySQL binlogs and outputs the changesets as JSON to Kafka. It's conceptually similar to databus and mypipe, but it can:

  • follow schema changes coming down the replication stream
  • output the changed rows as JSON
  • recover the binlog position where it left off -- maxwell stores no state on its client.

It's intended as an ETL tool and as a source for event-based services.
